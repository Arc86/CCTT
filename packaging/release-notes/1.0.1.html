CCTT 1.0.1

  • A real app icon. CCTT now has a proper rounded macOS icon in the Dock, Finder, and throughout the app.
  • The current logo everywhere. Onboarding, Settings, and the About page now show the up-to-date CCTT logo instead of older placeholder art.
  • Keychain prompt fixes. The access prompt now appears the moment you enable live limits (instead of a couple of minutes later), and CCTT caches the token so it no longer re-prompts every few minutes. Tip: choose Always Allow to avoid future prompts.
  • Renamed to CCTT.app. Fresh downloads install as CCTT.app; existing installs keep their current name and update in place.
